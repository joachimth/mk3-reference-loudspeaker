# Cabinet Build Notes

## External dimensions

- Width: 300 mm
- Depth: 370 mm
- Height: 1080 mm
- Material: 22 mm birch plywood
- Front vertical roundovers: R50

## Bass placement

- Woofer centers: approx. 520 mm from bottom (both sides, opposed at the same height)
- Configuration: side-mounted push-push, opposed + coupling block (see Chapter 8)

## Mid/tweeter layout

- Waveguide
- ScanSpeak 15W/4434G00 below waveguide
- Target c-c spacing: 140 mm
