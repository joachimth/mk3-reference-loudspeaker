# ScanSpeak 18W/4424G00 — Datasheet Reference

**Source:** [Official ScanSpeak product page](https://www.scan-speak.dk/product/18w-4424g00/) + [official datasheet PDF](./18W-4424G00.pdf) (Discovery series, datasheet updated May 18, 2020).

**Driver family:** Discovery series, 18 cm midwoofer, 4 Ω.
**Role in Mk3:** Midrange driver from **v9** (replaces the 15W/4434G00; see CHANGELOG v9 and DD-016).

---

## Mechanical dimensions (from official mounting drawing)

| Parameter | Value | Tolerance | Symbol / use in SCAD |
|---|---|---|---|
| Faceplate outer diameter | **179.20 mm** | ±0.30 | `flange_od` / `mid_face_d` |
| Faceplate / flange thickness | **5.2 mm** | — | `flange_tyk` / `mid_flange_t` (flush-mount rebate depth) |
| Basket / baffle cutout diameter | **144.3 mm** | +2 / −0 | `udskaering_dia` / `mid_cut_d` (cut Ø145.3 with 0.5 mm clearance) |
| Total depth (incl. front flange) | **72.2 mm** | — | `total_dybde` / `mid_depth` |
| Mounting BCD ("Pitch diam.") | **167.0 mm** | — | `bolt_cirkel` / `mid_bcd` |
| Mounting hole diameter | **5.3 mm** | ±0.15 (×6 holes, 60° apart) | `bolt_hul_dia` / `mid_hole_d` |
| Effective diaphragm diameter | 132 mm | — | (T-S only, not a mounting dim) |
| Cabinet displacement volume | **0.28 L** | — | `ts_displ` (datasheet "Mechanical Data") |

Screw land check: cutout Ø145.3 inside BCD Ø167 with Ø5.3 holes leaves
(167 − 5.3 − 145.3)/2 ≈ **8.2 mm** of wood inside the bolt circle — fine
(unlike the 15W, where this was the tight spot).

The parametric 3D model is `cad/midrange.scad` (driver, 2D baffle cutout,
drill template, half-section). `cad/cabinet.scad` imports every dimension
from it via exported functions.

## Thiele-Small parameters (from official datasheet)

| Parameter | Value | Unit | Use |
|---|---|---|---|
| Fs (resonance) | 49 | Hz | mid chamber sizing |
| Qms | 4.57 | — | — |
| Qes | 0.42 | — | — |
| Qts (total Q) | 0.38 | — | sealed chamber Qtc |
| Re (DC resistance) | 3.2 | Ω | amplifier match |
| Sensitivity (2.83V / 1m) | 91 | dB | level matching (SB26STAC is 91.5 dB → ~0.5 dB pad) |
| VAS (equivalent volume) | 24.1 | L | sealed chamber sizing |
| Sd (effective cone area) | 137 | cm² | displacement, SPL calcs |
| Bl (force factor) | 5.2 | T·m | motor strength |
| Xmax (linear excursion) | ±2.8 | mm | headroom above 150 Hz HPF |
| Power (IEC 18.4 / 18.2) | 50 / 110 | W | — |

**Datasheet box recommendations:** closed box **Vbox 13 L, f(−3dB) 85 Hz**;
vented 26 L / fb 48 Hz (not used — the Mk3 mid chamber is sealed).
A ~11 L sealed chamber gives Qtc ≈ 0.68, Fc ≈ 88 Hz (echoed automatically
by `cad/cabinet.scad`) — comfortably below the 150 Hz LR4 highpass.
